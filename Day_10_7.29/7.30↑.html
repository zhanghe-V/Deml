  • 3.var foo = {
        name: 'fooname',
       getName: function() {
        return this.name
          }
        }
        alert(foo.getName());

    答案： fooname
    this => foo
    当函数被保存为对象的一个属性时，成该函数为该对象的方法。函数中this的值为该对象。
